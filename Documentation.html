1.Introducere

1.1 Scop

Acest proiect vine cu o solutie pentru cei ce doresc atat o catalogare online a colectiei personale de produse filatelice cat si vizualizarea de alte colectii.

1.2 Audienta si sugestii despre continut

Acest document este destiant atat developerilor care in acest caz sunt studentii cat si profesorului de la laborator

In continuarea documentului sunt specificate principalele componente ale website-ului si interactiunea dintre ele. De asemenea sunt justificate alegerile facute in faza de proiectare.

1.3 Descrierea produsului

Stamp Worldn't este un proiect destinat celor ce doresc sa faca parte dintr-o comunitate internationala de colectionari de produse filatelice si are ca scop usurarea interactiunilor dintre acestia.

1.4 Referinte

Stamp World

2.Descriere generala

2.1 Perspectiva Produsului

Un webiste interactiv care contine informatii structurate sub forma mai multor pagini:

  • Home
  • Profile
  • Catalogue
  • FAQ
  • About US

2.2 Functiile Produsului

Principalele functii ale website-ului sunt:

  • Crearea unui cont
  • Adaugarea timbrelor in profilul propriu, acesta fiind public
  • Crearea de colectii
  • Vizualizarea colectiilor postate de alti utilizatori
  • Vizualizarea/descarcarea de statistici

2.3 Platforme operabile

Website-ul isi adapteaza automat formatul in functie de platforma de pe care este accesat(mobile sau desktop).

2.4 Design si implementare

Designul de baza este unul minimalist in care predomina nonculorile care sunt in corelare cu tematica continutului.

Fonturile folosite sunt:

  • Poppins
  • sans-serif

Butoanele se afla in contrast cu continutul paginilor pentru a putea fi observate usor. Acestea sunt amplasate in partea de colturile componentelor, acestea fiind usor distantate de continutul propriu-zis.

Tehnologiile folosite sunt :

  • HTML
  • CSS
  • JavaScript
  • PHP
  • SQL

3.Interfete externe

3.1 Interfetele utilizatorului

  • Front-end software: HTML, CSS, JavaScript
  • Back-end software: PHP,SQL

3.2 Interfetele hardware

  • Windows, MacOS, Linux, Android, IOS
  • Un browser care suporta HTML & JavaScript

4.Caracteristici de sistem

4.1 Crearea unui cont de utilizator

Pentru a putea accesa majoritatea functiilor disponibile este necesara autentificarea.

4.2 Crearea/editarea profilului propriu

Odata ce utilizatorul isi creaza un cont acesta isi poate edita datele personale accesand pagina "Profile"

4.3 Adaugarea produselor filatelice in baza de date

Produsele filatelice pot fi facute publice folosind meniul din pagina "Profile"

4.4 Crearea de cataloage

Utilizatorul poate crea cataloage in care sa introduca diverse produse

4.5 Cautarea avansata de produse

Utilizatorul poate cauta produse filatelice dupa anumite criterii cum ar fi titlu , tara, an, valoare, etc..

4.6 Vizualiarea/descarcarea de statistici

Utiliatorul are acces la diferite statistici legate de vizualizari, postari si activitate pe care le poate vizualiza direct pe site sau le poate descarca pe propriul dispozitiv.

4.7 Vizualiarea detaliata a produselor

Produsele pot fi accesate in mod individual, fiind afisate informatii detaliate despre acestea.
